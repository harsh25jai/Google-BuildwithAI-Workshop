# Google Doodle

This project was created as part of the **Google Build with AI RoadShows: Bengaluru 2026**.

## Project Setup

This project is built using [Vite](https://vitejs.dev/). To get started locally, follow these steps:

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm run dev
   ```
   This will start a local development server with Hot Module Replacement (HMR).

3. **Build for Production**
   ```bash
   npm run build
   ```
   The production-ready assets will be generated in the `dist` directory.

4. **Preview the Build**
   ```bash
   npm run preview
   ```
   This serves the production build locally for testing before deployment.
